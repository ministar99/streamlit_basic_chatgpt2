.gitIgnore 추가하기 (번외) 
먼저 .gitIgnore 을 작성해야 한다. 나의 경우 https://www.toptal.com/developers/gitignore/ 
사이트를 이용해 작성한다. 
그 다음 저장소에 파일을 올려야 하는데, 나는 github 에 직접 작성한 다음 로컬에 pull 을 
받는 방법을 사용한다. 
다른 방법으로는 로컬에서 만들어서 push 하는 방법이 있다. 여기선 로컬에서 push 하는 
방법을 설명한다. 너무 간단하다. 
위 사이트에서 파일을 만들어 로컬 저장소 최상위 루트에 .gitignore 를 위치시킨 다음
